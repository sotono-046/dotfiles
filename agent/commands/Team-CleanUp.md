# チーム掃除するコマンド

`/Team-Create` や `/Team-Create-Test` で編成したエージェントチームを片付ける。

1. 稼働中の teammate / 起動済み Task を確認する（`TaskList` 相当）。
2. 各 teammate に残タスクがないか確認し、未完了かつ再開不要なものは理由を添えて中断する（`TaskStop` 相当）。進行中で価値のあるものは完了を待つか、担当者に引き継ぐ。
3. 全 teammate が停止したことを確認してからチームを解散する。
4. 生成物（コミット、PR、ドラフトファイルなど）が残っている場合は一覧化し、ユーザーに引き渡す。放置されたブランチや未コミット変更があれば警告する。
5. 掃除が完了したら、何を止め・何を残したかを簡潔に報告する。
