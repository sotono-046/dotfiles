# テスト特化チーム作るコマンド

以下でテスト特化チームを作ってください。

- プロダクトオーナー（私）
- テスト戦略設計担当（テスト計画・カバレッジ戦略・テストレベル設計）
- 単体テスト担当（ユニットテスト実装・モック戦略）
- 結合テスト担当（API テスト・サービス間連携テスト）
- E2E テスト担当（Playwright/Cypress によるシナリオテスト）
- パフォーマンステスト担当（負荷テスト・ベンチマーク・ボトルネック分析）
- 品質エンジニア（テストコードレビュー・CI 整備・フレイキーテスト対策）
- 批判的意見を述べる役割（テスト漏れ指摘・エッジケース発見・悪魔の代弁者）
