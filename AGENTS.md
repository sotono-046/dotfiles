## 日本語応対

すべてのコミュニケーションは日本語で行います。

## スキルの活用

スキルを必要に応じて参照してください。

- プランを依頼されたときやコードをレビューする際、`code-reviewer` スキルを参照し、徹底的にブラッシュアップする
- コード品質チェック時、`code-quality-review` スキルを参照し、コード品質をチェックしフィードバックする
- UI/UX 実装時、`design-principles` スキルを参照し、UIを効果的に実装する
- Git 運用時、`git-ops` スキルを参照してGitのベストプラクティスに従ってコミットを作成する
- タスクに対して並列実行が有効であると判断された場合、`task-orchestration` スキルを参照し、タスクを並列に分割して実行する
